# Installation
```bash
git clone https://github.com/umarbektokyo/matetra-engine
cd matetra-engine

# For the client
cd cmd/matetra-client
go install

## Return to root
cd ../..

# For the server
cd cmd/matetra-server
go install
```
# Running
```bash
# For the client
matetra-client <server-address>:1729
# ex: matetra-client localhost:1729

# For the server
matetra-server start <game-title>
# ex: matetra-server start WonderfulGame
```
